# TransitFlow

Aplicación web para consultar trayectos y estado de transporte (autobús, tren y avión) desde una sola interfaz.

## Stack

- Frontend: React + TypeScript + Vite
- Estilos: Tailwind CSS
- Routing: React Router
- Backend (estructura inicial): Node.js + Express

## Estructura

- `src/components/`
- `src/pages/`
- `src/hooks/`
- `src/types/`
- `src/utils/`
- `src/context/`
- `src/api/`
- `server/src/routes/`
- `server/src/controllers/`
- `server/src/services/`
- `server/src/config/`

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```
