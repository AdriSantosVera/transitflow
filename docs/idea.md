# Idea del Proyecto – TransitFlow

## Descripcion general

TransitFlow es una aplicacion web orientada a la planificacion de viajes. Reune en un mismo panel los datos principales de cada viaje: destino, fechas, presupuesto, ahorro acumulado, gastos, lugares guardados, notas e itinerario basico.

La idea es tener una herramienta clara y visual para organizar un viaje antes de salir, sin depender de varias apps separadas.

---

## Problema que intenta resolver

Cuando preparo un viaje suelo repartir la informacion entre notas, webs, calculos sueltos y capturas. Eso hace dificil saber cuanto voy a gastar, cuanto llevo ahorrado y que lugares quiero visitar.

TransitFlow centraliza todo eso en una sola aplicacion para que el viaje tenga un seguimiento claro desde la planificacion inicial hasta el control del presupuesto.

---

## Usuario objetivo

La aplicacion esta pensada para:

- personas que organizan escapadas o vacaciones por su cuenta
- usuarios que quieren controlar presupuesto y ahorro por viaje
- viajeros que quieren guardar lugares, notas e ideas antes de salir

---

## Funcionalidades principales

- creacion de viajes
- edicion y borrado de viajes
- detalle de viaje por destino
- control de presupuesto
- registro de gastos
- registro de ahorro
- gestion de lugares por viaje
- gestion de notas por viaje
- favoritos
- dashboard general con metricas visuales

---

## Funcionalidades opcionales

- filtros por fecha o estado
- imagen destacada por viaje
- resumen visual con graficas
- ajustes visuales o de perfil

---

## Mejoras futuras

- autenticacion de usuarios
- compartir viajes o itinerarios
- checklist de viaje
- recordatorios
- documentacion de API con Swagger
- despliegue final con documentacion mas profesional

---

## Enfoque tecnico

Para el desarrollo del proyecto se utiliza una arquitectura basada en frontend y backend:

- Frontend: React con TypeScript
- Estilos: Tailwind CSS
- Backend: Node.js con Express
- Persistencia: Prisma + SQLite
- Comunicacion: API REST

El objetivo es mantener una aplicacion visualmente cuidada pero con una base tecnica real y defendible.

---

## Conclusion

TransitFlow ha evolucionado de una idea inicial simple a una app funcional de planificacion de viajes con persistencia real. La base actual permite gestionar viajes, ahorro, gastos, lugares y notas dentro de una sola interfaz moderna.
